# Boussinesq-stress-distribution
Stress distribution according to the classical Boussinesq problem
Ниже представлен готовый код, который рассчитывает распределение напряжений по классической задаче Буссинеска (вертикальное давление от точечной силы на поверхности) и строит «луковицу напряжений» с изолиниями и цветовой шкалой.
